SaaS Invoice Management System
Overview

This is a web-based SaaS application for managing clients, invoices, and payments. Developed during my internship (March – April 2026) as a full-stack project using Node.js, Express, PostgreSQL, React.js, and TailwindCSS.

The system allows users to:

Manage clients

Create and manage invoices

Track payments

Generate revenue reports

Visualize business performance through dashboards

Features

Authentication & Security: User registration, login, JWT authentication, bcrypt password hashing, multi-user data isolation

Client Management: CRUD operations, search/filter

Invoice Management: Create invoices, add items, edit/delete, status tracking (draft/pending/paid), automatic total calculation

Payment Management: Record payments, update invoice status, track remaining balance

Dashboard & Reports: Revenue statistics, pending payments, top clients, monthly revenue chart

Optional Features (time permitting): PDF invoice export, email notifications, password reset

Tech Stack

Backend: Node.js, Express.js, PostgreSQL, JWT, bcrypt

Frontend: React.js, TailwindCSS, React Router, Chart.js / ApexCharts, Axios

Database: PostgreSQL relational database with multi-tenant support
